# ExpenseVision AI

## Features
- AI-Powered Insights: Predictive analytics flag overruns before they happen.
- Real-time Analytics: Live expense tracking with interactive dashboards.
- Smart Budgeting: Category-level budgets with threshold alerts.
- Board-ready Reports: Generate stunning summaries easily.
- Team Collaboration: Role-based access for organizers and finance leads.

## Installation
Run the following command to install the project dependencies:
```bash
npm install
```

## Running the project
Start the local development server:
```bash
npm run dev
```

## Build instructions
Build the project for production:
```bash
npm run build
```
